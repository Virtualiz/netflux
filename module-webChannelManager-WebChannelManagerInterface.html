connectWith(wc, id, jpId, peerIds)

Send a request to a peer asking him to establish a connection with some peers. This function is used when a new peer is joining the WebChannel. The request can be sent to the peer who is joining as well as other peers who are already members of the WebChannel.

Parameters:
Name Type Description
wc WebChannel

The Web Channel.
id string

Id of the peer who will receive this request.
jpId string

Joining peer id (it is possible that id=jpId).
peerIds Array.<string>

Ids of peers with whom id peer must established connections.
