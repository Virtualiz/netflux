Module: webChannelManager

Web Channel Manager module is a submodule of module:service and the main component of any Web Channel. It is responsible to preserve Web Channel structure intact (i.e. all peers have the same vision of the Web Channel). Among its duties are:

  • Add a new peer into Web Channel.
  • Remove a peer from Web Channel.
  • Send a broadcast message.
  • Send a message to a particular peer.
