Class: ChannelProxyService

channelProxy~ ChannelProxyService

This is a special service class for ChannelInterface. It mostly contains event handlers (e.g. onmessage, onclose etc.) to configure a newly created channel. Thus be careful to use this in handlers, as it will refer to the instance of ChannelInterface and not to the instance of ChannelProxyService.
